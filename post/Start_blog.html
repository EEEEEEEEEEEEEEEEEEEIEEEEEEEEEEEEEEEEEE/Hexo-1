从博客谈起

发表于 2017-04-22 | 分类于 笔记

　　使用博客，我的意图在于知识的汇总和分享。知识汇总是重复的琢磨学习，分享是别人能从自己汇总的知识中获得的成就感。

　　博客的起因，16年因为痴迷Dropbox ，Dropbox 一个衍生的功能就是搭建博客，所以才有了第一次尝试搭建博客。因为Dropbox 国内访问不了，博客国内没法访问，再加上也不怎么会搭建，就是挂代理也经常打不开网页。之后认识了Farbox 简易型的博客搭建，博客的模板只有几个，选择模板，绑定网址，几乎是一键式的搭建。这种博客的意义在于个人可以专注于写文章。期间博客被我冷落了很长一段时间，直到最近转战到Github 搭建博客。

　　Farbox 搭建的简易型，不太符合我个性化的需求。太过于简单。我的性格适合不停地折腾，所以Github Hexo 博客应运而生。Github 起初只是用来信息的发布，在Github 页面上面。后来在朋友的鼓动下，搭建了博客，琢磨了几天时间，加上后期的完善，10多天的时间搭建成了现在的Hexo 博客。各项功能都还算满意。

　　昨天尝试了Wordpress 搭建博客，比Hexo 博客更个性化，但问题在于必须需要服务器的基础。使用Wordpress 还是Hexo 。我考虑的着重点是数据的长期保存。不在乎买服务器的费用，而在于数据的长期保存。在服务器上搭建，服务器不缴费数据就会丢失，而在Github 上搭建Hexo ，只要Github 这家公司在，我的博客都会一直在。所以打算一直使用Github 搭建的Hexo 博客。使用其他的也只是暂时的学习技术。

　　搭建的同时，问题很多，毕竟我没有学过程序。感谢开源，感谢百度，感谢谷歌，也要感谢Hexo 和 Github ，还有很多互联网上教程的作者和群里给我耐心指点的朋友。
