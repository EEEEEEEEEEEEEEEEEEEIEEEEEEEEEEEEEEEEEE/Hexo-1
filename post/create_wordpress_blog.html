建立免费的Wordprss博客

发表于 2017-06-01 | 分类于 教程

　　相对于Hexo博客，wordprss 博客更为简单，一个手机App就可以搞定。通过手机开通，搭建，选择主题，撰写文章。对于新手以及经常使用手机的朋友是个不错的选择。
　　
　　Wordpress App可以帮助你完成这些，同时它支持IOS，安卓以及桌面版本，并提供免费套餐。免费套餐使用二级域名，无法更改域名，以及3G的网络空间，相信对可以满足大多数人的空间要求，毕竟是免费。如果你想拥有自己的域名或者想要用自己的域名，就只能用付费版的套餐。向前面说的那样，免费版的足够我们使用。

下载Wordpress 并安装手机App

注册Wordpress 账号

点击创建站点


值得注意的是，这里的站点地址就是我们网站地址的前缀，创建好之后，无法修改。

修改主题，添加菜单，添加页面

　　就这样，你就拥有了自己的博客网站，并且可以在自己的手机上部署完成各种各样的工作。
　　
　　有一些功能是手机无法完成的，比如屏蔽评论等功能。我们必须借助电脑完成这类的操作。登录网站https://Wordpress.com。登录我们注册的账号，进入菜单进行修改。

教程完毕
